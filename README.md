# terminal-ui-crafting

A reusable agent skill for designing and refining terminal interfaces.

This skill now emphasizes:
- workflow-first terminal UX
- choosing the right stack for the job
- respecting existing repo conventions
- avoiding canned themes and one-size-fits-all layouts
- validating the real terminal entrypoint

## Install with skills.sh / Vercel Skills

```bash
npx skills add schin-300/terminal-ui-crafting-skill
```

## Install the specific skill

```bash
npx skills add schin-300/terminal-ui-crafting-skill --skill terminal-ui-crafting
```
