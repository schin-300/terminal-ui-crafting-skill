# terminal-ui-crafting

A reusable agent skill for designing, implementing, and refining stylish terminal UIs and TUIs.

This skill focuses on:
- Textual/Rich operator dashboards
- restrained neon / operator-deck aesthetics
- boot splashes, hotkeys, and tabbed workflows
- smoke-tested terminal-first delivery

## Install with skills.sh / Vercel Skills

```bash
npx skills add schin-300/terminal-ui-crafting-skill
```

## Direct skill path

```bash
npx skills add schin-300/terminal-ui-crafting-skill --skill terminal-ui-crafting
```
